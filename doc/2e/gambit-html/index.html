Gambit ¶

Gambit is a complete implementation of the Scheme language aimed at portability, speed, and power.

For the most up-to-date information on Gambit and related resources, visit the Gambit home page. Please report issues via the Gambit repository on GitHub.

This is the Gambit Scheme manual, revised edition. It documents Gambit v4.9.5 (September 20, 2024).
